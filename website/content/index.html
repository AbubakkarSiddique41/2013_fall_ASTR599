Course Information

ASTR 599: Scientific Computing with Python

University of Washington; Fall 2013


Instructor: Jake Vanderplas

Meeting Times:

  • Bootcamp: 9:00am - 4:00pm, September 20-21
  • Seminar: 3:00pm-4:00pm, Mondays

Description

This graduate seminar course will offer a comprehensive introduction to scientific computing in Python, geared toward students and researchers in Astronomy. It consists of a 3-hour boot camp before the start of the quarter, followed by a weekly hour-long seminar covering more specialized topics.

Much of the curriculum will be modeled after the Berkeley Python Bootcamp.
